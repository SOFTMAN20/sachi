## Troubleshooting

### "Account Not Found" Error
**Solution:** You need to login to Expo Go with the `softman20` account

### "Installation Blocked"
**Solution:** Enable installation from unknown sources in Android settings

### "App Not Installed"
**Solution:** 
- Make sure you have enough storage
- Uninstall any previous version
- Try downloading again

### "Parse Error"
**Solution:** 
- Download might be incomplete
- Try downloading again
- Make sure you're on Android 5.0 or higher
